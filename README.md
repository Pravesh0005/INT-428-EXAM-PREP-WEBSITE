## 🧠 Unit Breakdown

<details>
<summary><b>Unit 1 — Foundations & Applications of AI</b></summary>

- What is Intelligence vs Artificial Intelligence
- **6 Foundations** of AI (Philosophy, Math, Psychology, CS, Neuroscience, Linguistics)
- **Types of AI**: Narrow (EXISTS), General (Theoretical), Super (Theoretical)
- Expert Systems — Knowledge Base + Inference Engine + Certainty Factors
- Production Systems — "A system of rules for problem-solving"
- **Key Trap**: Siri/Alexa = Personal Voice Assistants (NOT ML algorithms)
- **Key Trap**: Astrology is NOT a foundation of AI

</details>

<details>
<summary><b>Unit 2 — Problem Solving & Search Algorithms</b></summary>

- State Space, Branching Factor, Problem Formulation
- **Uninformed Search**: BFS (Queue), DFS (Stack), IDS (Complete + Optimal)
- **Informed Search**: Greedy `f(n)=h(n)`, A* `f(n)=g(n)+h(n)`
- **Admissibility**: h(n) ≤ h*(n) — never overestimates
- Minimax, Alpha-Beta Pruning (reduces nodes evaluated)
- Simulated Annealing (accepts worse solutions to escape local minima)
- Water Jug Problem (states = (cap1+1)×(cap2+1))
- De Morgan's Laws, Propositional Logic

</details>

<details>
<summary><b>Unit 3 — Machine Learning</b></summary>

- Supervised vs Unsupervised vs Reinforcement Learning
- **Confusion Matrix**: Precision=TP/(TP+FP), Recall=TP/(TP+FN)
- **Standard Deviation** numericals (SD of {3,7,7,7,9} = 2)
- **Bayes Theorem**: P(A|B) = P(B|A)P(A)/P(B)
- **Naive Bayes** assumption: feature independence given class
- **KNN** numerical classification
- **K-Means** 1-iteration problems
- Cross-validation = reduce overfitting
- Reinforcement Learning MDP components

</details>

<details>
<summary><b>Unit 4 — Deep Learning & NLP</b></summary>

- Perceptron (linear), MLP (non-linear, XOR solver)
- **CNN**: Conv → Pool → FC. Spatial data. Pooling = reduce dimensions
- **RNN**: Sequential data. Vanishing gradient = KEY problem
- **Transformer**: Parallel processing. Self-Attention Q,K,V. Positional Encoding
- **BERT**: Bidirectional, MLM+NSP, Encoder-only, Understanding tasks
- **GPT**: Autoregressive, Next-token, Decoder-only, Generation tasks
- NLP Pipeline: Speech Recog → Text Preprocess → Intent Recog → Response Gen
- Edit Distance for spell checking (NOT TF-IDF)

</details>

<details>
<summary><b>Unit 5 — Generative AI & Prompt Engineering</b> ⭐ Inferred</summary>

- **GAN**: Generator (creates) + Discriminator (judges). Generator = generates samples
- **VAE**: Probabilistic latent space. Primary purpose = generate new data
- **Diffusion Models**: Add noise forward, remove noise backward
- **LLMs**: Emergent abilities, zero-shot, few-shot
- **Prompt Patterns**: Instruction, Chain-of-Thought, Few-shot (ALL are valid)
- **Zero-shot**: Predict unseen tasks directly
- AI Ethics: Bias, Fairness, Transparency, Hallucination

</details>

<details>
<summary><b>Unit 6 — Data Analysis, Tools & MLOps</b> ⭐ Inferred</summary>

- **ChatGPT ADA**: Can analyze datasets + run Python (standard ChatGPT cannot)
- **Visualization**: Categorical → Bar chart. Two variables → Scatter. Trends → Line
- **Python pandas**: `df.drop_duplicates()`, `df.fillna()`, `ZeroDivisionError`
- **MLOps**: Automates training, deployment, monitoring
- **Tools**: Grammarly (writing AI), ChatGPT (marketing), Tableau (visualization)
- Knowledge Representation: Frames, Semantic Networks, Logical Representation

</details>

## ⚠️ Top MCQ Traps (Memorize These!)

```diff
- WRONG: Greedy BFS is optimal → CORRECT: Only A* is optimal
- WRONG: Vanishing gradient is CNN problem → CORRECT: It's RNN's problem
- WRONG: BERT reads left-to-right → CORRECT: BERT is BIDIRECTIONAL
- WRONG: Generator judges fake/real → CORRECT: DISCRIMINATOR judges
- WRONG: Astrology is AI foundation → CORRECT: It is NOT
- WRONG: Siri = ML algorithm → CORRECT: Personal voice assistant
- WRONG: df.drop_duplicates() for NA → CORRECT: df.fillna() for NA
- WRONG: Branching factor = paths to goal → CORRECT: States from given state
```
